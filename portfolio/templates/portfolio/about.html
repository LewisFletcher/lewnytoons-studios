Site Background

When I began building this website, my intention was to create a platform which I could use to share my music with the world. This turned from a small project into a passion for coding, transforming a small website into so much more. As I am most familiar with Python, I chose to build this website with Django, along with Tailwind's CSS for styling purposes. I utilized Django's database features to design a database for the upkeep of the blog, merch, and music sections of this website. This website has no CRUD interface as the only user posting is myself, through the administrator section. I am very familiar with CRUD applications and am more than capable of adding this feature to a site (with or without a required login).
